# Option-Critic in pinball domain
## Requires
* python == 3.6+
* gym
* [gym_pinball](https://github.com/Kerneltyu/gym-pinball)
* numpy
* scipy
* matplotlib
* tqdm
* pandas

## How to run
```
cd src
python oc_agent.py
```

### With visualization
```
python oc_agent.py --vis
```

### Read Model files
```
python oc_agent.py --model data/20190301/2211/model
```
